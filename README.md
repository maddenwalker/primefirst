# PrimeFirst
Delivery Slot Alert for Amazon Prime Now

- Create and edit `.env` file like `.env.local` (with your Amazon login)
- Run commands bellow,

```
npm install
npm run build
npm run start
```

- It play the alert notification at initialization
- You need a prepared cart on your account
